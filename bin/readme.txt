1. Установка и запуск:
Продукт портабельный и не требует установки, запуск осуществляется при помощи файла englishsimulator.exe.

2. Взаимодействие с программой:
Программа не имеет графического пользовательского  интерфейса, все взаимодействие сводится к вводу необходимых команд с клавиатуры в текстовую строку консоли. Новый текст, при помощи которого программа общается с пользователем, также появляется в окне консоли. Читать окно консоли можно при помощи плоского просмотра в NVDA или jaws курсором в jaws соответственно.
Внимание! Программа практически не защищена от некорректных команд. Ничего страшного не произойдет, но с какой-то долей вероятности, приложение просто аварийно завершит свою работу. Для продолжения работы с приложением, достаточно просто запустить его заново.
Все команды вводятся маленькими буквами, всегда проверяйте корректность введенных команд.
Выход из программы осуществляется при помощи клавиш CTRL+c. Если этот вариант не работает, вызываем меню при помощи alt+space и активируем пункт меню "выйти".

3. Описание пунктов меню:
Главное меню имеет несколько пунктов.
Пункт №6, помогает выбрать и настроить sapi5 синтезатор, который будет озвучивать английские слова/фразы;
Пункт №5, помогает создать новую базу для слов/фраз;
Пункт №4, помогает добавить слово или фразу в базу, список баз будет представлен после выбора пункта. Для выбора базы, достаточно ввести ее номер;
Пункт №3, запускает режим опроса слов, в списке представленных баз, можно ввести одновременно несколько номеров через пробел, тогда будут загружены слова/фразы из указанных баз;
Пункт №2, включает изучение неправильных глаголов, база выбирается все тем же вводом номера базы из представленного списка;
Пункт №1, включает режим изучения слов, где слова/фразы будут представлены на двух языках.

4. Дополнительные опции:
Режимы изучения слов и неправильных глаголов, имеют следующие опции:
Режим изучения слов, предлогает выбрать направление перевода для опроса, по умолчанию установлено "ru", Т.Е. слова будут даваться на английском языке, ответ необходимо писать на русском. По умолчанию, значит, что можно просто нажать ввод, не вводя явного значения.
Режим изучения глаголов, спрашивает, на каком языке задавать вопрос. По умолчанию выбран "en". При этом будет выводиться первая форма "Инфинитив" неправильного глагола, затем последовательно будут спрашиваться все формы данного неправильного глагола. Если выбрать "ru", глагол будет показываться на русском языке, затем также будет необходимо ввести все формы неправильного глагола.

Опция "repeat", уточняет, сколько раз повторять вопрос в случае неверного ответа, по умолчанию стоит "3".
Также будет предложен порядок опроса - 3 способа. Перечисление слов/фраз в порядке, как они записаны в базе, в обратном порядке или рандомно.
Следующая опция предоставляет возможность включения sapi5 синтезатора, которым будут произноситься английские слова/фразы, по умолчанию опция отключена.

Опция режима изучения слов "Study", - Включить аудио озвучивание слов/фраз, по умолчанию отключена и имеет варианты только аудио, аудио плюс текст и собственно только текст, что и выбрано по умолчанию.
Внимание!!! Эта опция вполне рабочая, но выбирать ее можно только в случае специально приготовленной базы, на момент написания этого ридми - это база different01. Всвязи с большими энергозатратами по подготовке подобных баз, автор и идейные вдохновители решили обходиться sapi5 синтезатором, но в последствии несколько баз с Нным набором слов, таки будет подготовлено.
